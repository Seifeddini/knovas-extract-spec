# Golden corpus — Markdown

**Target coverage**:
- `plain.md` — paragraphs and headings; `sections[]` populated from `#`/`##`/`###`.
- `frontmatter.md` — YAML frontmatter; keys lifted into `metadata` (title, author, language) where they match.
- `code-fences.md` — fenced code blocks preserved verbatim.
- `tables.md` — pipe tables; text representation per format convention.
