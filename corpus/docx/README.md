# Golden corpus — DOCX

**Target coverage**:
- `simple.docx` — plain paragraphs.
- `headings.docx` — exercises `content.sections[]`.
- `footnotes-comments.docx` — verifies that footnotes/comments are extracted but kept separate from body text.
- `embedded-image.docx` — image bytes ignored; alt-text captured in extra.
- `cjk.docx` — Asian script for encoding regression coverage.
